문서 하위 항목 사용 표기

DOM(Document Object Model)

        웹 문서의 모든 요소에 JavaScript가 접근하기 위해 설정한 소식
        요소 : <태그>내용물<태그>
        
        특정 태그 내에 작성된 아이디값이나 클래스값으로 접근할 수 있다.

        예)
        document.getElementById("id속성명");
        document
            - getElementById
                    -value
                    -style
                    -innerHTML
            - getElementByClassName
                    -length(글자길이 가져올 때 사용)
                    -item
                    -nameItem
            - getElementByIdTagName
                    -length(글자길이 가져올 때 사용)
                    -item
                    -nameItem
        
        문서 내에서 일치하는 아이디명을 가져오고
                    아이디명 내에서 값을 가져오겠다.
        document.getElementById("id명")
                getElementById("id명").value

        문서 내에서 일치하는 아이디명을 가져오고 값을 가져오겠다
        document.getElementById("id명").value
