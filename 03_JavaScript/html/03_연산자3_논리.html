논리 연산자 (&&,||,!)

AND 연산자 (A && B)

        
        -A, B 모두 true 일 경우 결과로 true를 변환하는 연산자
        (나머지 경우는 모두 false)
        A = 조건 1 B = 조건 2
        A  와  B가 모두 참일때만 true
        (내가 회원가입 할 때 작성한 값이 로그인을 시도했을 때)
        A = 아이디가 일치하는가 ?
        B = 비밀번호가 일치하는가 ?

로그인 하기

        -회원가입 할 때 저장한 아 이 디: 'member01'
        -회원가입 할 때 저장한 비밀번호: 'pass01'
        아이디와 비밀번호가 일치할 경우 alert("로그인 성공~")

        아이디와 비밀번호가 일치하지 않으면
        alert ("아이디 또는 비밀번호가 일치하지 않습니다.")

OR 연산자(||)

        
          -A, B 둘중 하나라도 true 일 경우 결과로 true 반환하는 연산자
          (A, B 모두 false일 경우에만 false)
          둘중 하나만 참이어도 참으로 혀용
