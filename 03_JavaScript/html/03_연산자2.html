복합 대입 연산자(+=, -=, *=, /=, %=)

        -기존 변수에 저장 되어있는 값에 새롭게 대입되는 값의 산술 연선 결과 대입

        (증감 연산자는 1씩 증가 감소, 복합 대입 연산자는 원하는 만큼 증가, 감소)

        ex)
        let temp = 10;
        temp += 5; // temp = temp + 5 
        결과 : temp 총 15로 변경

현재 값 : 0

+= (더하기 연산자)

      기존 값에 새로운 값을 더하거나 이어붙여 업데이트 하는 연산자
      == 동등 연산자 → 타입을 알아서 수정 / 수정이 안될 때 있음
      += 
      글자+글자 이어붙이거나
      숫자+숫자 누적시켜서 숫자크기를 키울 수 잇음

      let A = :"안녕";
      let B = "반가워";
          A+B → 안녕반가워 안녕 반가워를 붙여버리기

      let C = "5"
      let D = "5"
        C+=D → 10;
      let  E  =  "5"
      let  F = 5;
      E+=F → 결과가 10이 나올 수 있고, 55가 나올 수 있음
