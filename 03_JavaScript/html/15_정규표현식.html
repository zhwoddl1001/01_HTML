정규표현식 (RegularExpression, RegExp)

      -정규 표현식 또는 정규식
      -특정한 규칙을 가진 문자열 집합을 표현하는데 사용하는 형식 언어
      -입력된 문자열에 대한 특정 조건 검색, 일치 여부 판단, 치환에 대한
      조건문을 간단히 처리할 수 있음

정규 표현식 객체 생성 및 확인하기

      1. 정규 표현식 객체 생성방법
        1)const regExp = new RegExp("정규표현식");
        2)const reExp = /정규표현식/;

        2. 문자열 패턴 확인 기능(메서드 = 함수)
         1) regExp.text(문자열)
          -> 문자열에 정규표현식과 일치하는 패턴이 있을 경우 true
                                                없을 셩우 false 반환

        2)regExp.exec(문자열)
          ->문자열에 정규표현식과 일치하는 패턴이 있을 경우
              처음에 매칭되는 문자열을 반환
              없으면 null 값을 반환

정규표현식으로 표현하기

      문자열의 패턴을 나타내는 문자
      문자마다 지정된 특별한 뜻이 담겨있다.

      일반 문자열 : 같은 문자열이 존재하는지 검색
      -> /abc/ abc 라는 문자열이 존재하는지 검색

      -----------------------------------------------

      [] : 대괄호 안의 어떤 문자와 일치
      -> /[abcd]/ a,b,c,d 중 하나라도 일치하는 문자가 있는지 검색

      [0-9] : 0부터 9까지 모든 숫자 사용가능
      [a-z] : 모든 영어 소문자
      [A-Z] : 모든 영어 대문자
      [ㄱ-힇] : ㄱ부터 힇까지 모든 한글
      [가-힇] : ㄱ부터 힇까지 모든 한글(모음,자음 제외)

      --------------------------------------------------------

      [^] = 대괄호 안의 문자와 일치하지 않음
      ->[^abcd]a,b,c,d 중 하나라도 일치하는 문자가 없음

      ^(캐럿) : 문자열의 시작을 의미
      /^a/  == a로 시작
      
      $(달러) : 문자열의 끝을 의미
      /a$/ == a로 끝

      ---------------------------------------------------------------
      Escape 문자
      -특정 문자를 특별한 용도로 사용하려는 경우에 사용되는 문자
      
      1) \ + 특수문자 : 기능이나 패턴이 아닌 일반 문자 그대로 인식
      -> \^  : 문자열이 시작한다는 의미가 아니라 ^ 생긴 특수문자가 존재하는지 확인

      2) \d (digit, 숫자) : 아무 숫자(0~9 중 하나)
         \w (word, 단어) : 아무 문자(대신 띄어쓰기나 특수문자나 한글 불가)
         \s (space, 공간) : 아무 공백 문자(띄어쓰기, 엔터 탭 등 ok)

         ---------------------------------------------------------------
         *수량 관련 문자 
         a{5} : a문자가 5개 존재하는가
         a{2,5} : a가 2이상 5개 이하
         a{2,} : 2개 이상
