함수(function)

        - 목표하는 결과를 만들기 위한 코드의 묶음

        - 코드를 {} (중괄호, 블록)으로 묶어 작성하고
            붙여진 이름을 호출해서 사용
        
        * 함수 선언
        * 함수 정의
        * 함수 호출

        [기본 함수 형태]
        function 함수명() { //함수의 이름 선언
            //함수 정의
        }

        함수명(); //함수를 호출

        함수란 ? 특정적인 기능을 동작하고자 할 때 
                특정 기능을 묶어서 만드는 명칭
        
        예 )  
        
        function 전자레인지 작동시작("몇 초 인가요?"){
            1. 전자레인지 코드를 꼽는다.
            2. 전자레인지 ("몇") 초를 선택한다.
            3. 전자레인지를 ("몇")초만큼 실행한다.
            4. 전자레인지 ("몇")초가 다 되면 종료한다.
        }

매개 변수(Parameter)

        * 媒介變數(매개변수, parameter)
        - 함수 선언 시 () 안에 작성되는 변수
            -> 전달 인자로 부터 받은 값을 저장
            -> 함수 정의 부분에서 사용 가능

        媒 중매   매  중개하거나 연결한다
        介 개입할 개  무엇인가 사이에 끼어있는 상태
        變 변할   변  변화하고 바뀐다는 의미
        數 셀     수  수 숫자 변수 라는 의미
        
        Parameter : 변경이 될 기준을 제공
        para- : 근처, 옆에 제공하다
        meter : 기준

        중간 역할을 하거나 전달하는 변수

전달 인자(Argument)

        * 傳達 引子(전달 인자, Arguments) //자바에서 설명
        傳 전할     전 정보를 전하다
        達 도달될   달 전달의 완료 또는 목적지에 도착했다
        引 끌어당길 인 무언가를 특정 위치로 이끈다
        子 아들     자 특정한 목적이나 역할을 가진 요소
        
        Argument : 제공되는 정보, 실제 값
        함수(function)가 호출 될 때 아규먼트의 값들이 증거처럼 파라미터에 전달

            * 요소(Element) = <태그>요소값</태그>

무한한 결과가 아닌, 선택지가 있는 결과를 작성할 때 매개변수와 전달인자를 사용 매개변수와 전달인자는 한 세트 매개변수는 전달인자를 전달해주는 전달 역할일 뿐 전달인자는 위에서 선택된 전달인자가 들어옴 고정적X 매개변수 전달인자 function 결정사( let 소개팅대상 = "홍길동";){ }

return (반환)

        - 현재 수행중인 함수를 종료하고 호출한 곳으로 돌아감

        - 호출한 곳으로 돌아갈 때 
            return 옆에 작성된 값을 가지고 돌아갈 수 있다.

        
        만약에 return 100; 호출한 곳으로 100을 가지고 돌아갈 수 있음
