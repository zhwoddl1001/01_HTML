        -자바스크립트 모든 기능에서 가장 최우선

        -window의 자식 객체
        1)DOM(Document Object Model) : document
            (브라우저에서 내용(페이지) 출력 부분)

윈도우의 기능임을 생략해도 되는 기능들

        생략가능
        (window.)alert("글자")
        (window.)prompt("글자")
        (window.)setTimeout("함수,지연시간")

앞에 윈도우를 붙여줘야하는 기능들

        window.location.href = "이동할페이지.html"

        window를 붙여주는 것이 좋음

**href 와 src 차이
        링크나 리소스를 참조할 때 사용하지만 사용용도가 다름
        [href] = 페이지 이동이나 기능을 참조할 때 사용
                css파일,페이지 간 이동, 외부 웹 페이지 링크 등

        [src] = 페이지 안에 이미지, 동영상, Js 파일 등 외부소스를 삽입할 때 사용
                js파일, 이미지, 동영상 등
