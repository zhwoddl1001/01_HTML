입력된 키 확인

        키보드 자판을 누르고 있을 때, 누르고 있는 상태를 종료시킬 때
        
        이 현상을 키 입력 이벤트 JS에서 키 입력 이벤트를 처리하는데 사용되는 이벤트 리스너
        각 이벤트는 키가 눌리거나 떼어질 때 발생

        이벤트 종류와 차이점
        onkeydown : 사용자가 키를 누르는 순간 발생
        onkeyup : 사용자가 키에서 손을 떼는 순간 발생
        onkeypress : 사용자가 키를 누르고 있을 때 발생
                    (현재 대부분 인터넷에서는 사용을 권장하지 않음)
