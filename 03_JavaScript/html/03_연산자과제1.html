비교 및 연산 예제

논리 부정 (NOT) 연산자(!)

        -논리값을 반대로 바꾸는 03_연산자과제
        - 청개구리
        
        값이 false일 때 true

        예를 들어
        const 사과 = "apple";
              true
        사과 != "apple" ? "사과가 apple 이 아니라 grape 입니까?" 일 때 보이는 곳  
                        :  "여기에 값이 보임";

부등호 (=, == ,===)

        = (할당 연산자)
          -변수에 값(숫자나 문자)를 할당할 때 사용
          -ex) const a = 5;   let 사과 = "빨간색";

        == (동등 연산자)
            - 값만 비교하는 연산자. 타입이 다르더라도 값이 같은면 true 반환
            -ex)
                const 숫자5 = 5;  
                const 글자5 = "5";      
                숫자5==글자5 가 같습니까? 답:같다
                    값을 우선적으로 보기 때문에 숫자나 형식의 값이 일치하면 같다고 표현
        === (일치 연선자)                    
            - 값과 타입이 모두 일치하는지 비교 값과 모두 같을 때만 true 반환
            - ex)
                const 숫자5 = 5;  
                const 글자5 = "5"; 
                숫자5와 글자5가 같습니까> ===> 하나는 문자고 하나는 숫자여서 다르다? 답:false
                동등 연산자보다 엄격한 비교를 원할 때 사용

        == 는 타입 변환이 자동으로 일어나 일어나는 기준을 알 수 없음
        ===는 타입 변환을 자동으로 일어나지 못하게 설정 후 비교
