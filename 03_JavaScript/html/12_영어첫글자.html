첫글자가 대문자로 시작하는 자바스크립트

        Math
        Array
        Date와 같이 영어로 시작하는 글자의 첫 글자가
        대문자로 시작하는 단어는 자바스크립트에서 파일을 만들어서 기능을 제공하거나
        개발자가 외부에서 사용할 수 있는 파일을 만들어서 외부에서 사용 가능하도록 표기하는것

        주로 ReactJs, Java에서 많이 사용하는 단어 특징
        Math
        Array
        Date
        자바스크립트 내에서 개발자들이 편리하게 사용할 수 있도록 만든 기능
        개발자의 눈에는 보이지 않지만
        Math.js array.js Date.js 와 같은 파일로 만들어진 기능을 가져다 쓰겠다는 표현

        파일명 자체에서 첫 글자를 대문자로 작성해준다는 것은
        외부에서 사용하겠다는 내적 의미가 포함

        파일명 자체에서 첫글자를 소문자로 작성해준다는 것은 외부에서 사용하지 않고 내부에서만
        사용하고 말겠다는 의미
