확장프로그램 종류

    1.Auto Rename tag           : 태그명이 변경될 때 뒤에 따라오는 태그도 자동으로 변경
    
    2.Code Spell Checker        : 코드에서 철자나 문법이 문제가되어 에러가 생기지 않도록 도와줌
    
    3.Highlight Matching Tag    : 태그가 제대로 닫아졌는지 순서는 맞는지 확인해줌
    
    4.Image Preview             : 코드 내에서 이미지 위에 마우스를 살짝 올리면 이미지 미리보기 가능
    
    5.Korean Language Pack      : 한국어로 vscode 설정을 변경
    
    6.Live Preview              : HTML에 작성한 코드와 디자인을 미리보는 확장 프로그램
    
    7.Prettier - Code formatter : 정렬할 때 개발자 형식에 맞게 정렬해주는 확장 프로그램 단축기 Shift + Art + F
