display flex - justify-content

      justify-content : div(=container)와 같은 태그를 수평 방향으로 정렬하고 
      간격을 조절하는 데 사용
      display:flex 를 작성하면 justify-content:flex-start 이미 존재함
      거의 모근 태그 justify-content:flex-start
      justify-content:flex-start = 요소들의 컨테이너의 왼쪽 정렬 (기본값)
      justify-content:flex-end   = 요소들이 컨테이너의 오른쪽 정렬
      justify-content:center     = 요소들이 컨테이너의 가운데 정렬

      태그들의 특정적인 규칙을 제공한 상태로 정렬할 때 사용하는 방법
      justify-content:space-between = 첫 번째 요소는 왼쪽끝 마지막 요소는 오른쪽 끝 나머지 요소들은 균등한 간격을 유지
      |요소1        요소2        요소3        요소4|
      
      justify-content:space-around  = 각 요소 사이 동일한 간격 양쪽끝에는 반 간격씩 제공
      |   요소1      요소2      요소3      요소4   |
      
      justify-content:space-evenly  = 모든 요소 사이와 양쪽 끝에 동일한 간격을 제공
      |     요소1     요소2     요소3     요소4     |

가로로 정렬

display:flex + justify-content:center

display:flex = 세로로된 div를 가로로 보기 justify-content:center = 가로에 있는 div를 가운데 정렬
