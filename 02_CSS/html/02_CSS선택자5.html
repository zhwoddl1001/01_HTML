태그 속성 선택자

label 태그

        속성 선택자 css 문법
        for 속성의 값이 input 요소와 연결하는 역할
        input에서 id값과 for와 속성값이 같으면 연결
        또는
        label안에 input 태그를 작성할 수 있음
        label안에 input 태그를 작성하면 for는 생략 가능

label 예제 작성

파일 선택 버튼을 누르지 않고 [사진선택하기] 를 선택하면
파일 선택하는 창이 연동돼서 열리게 됨
주로 input type="file" 속성은 display:hidden을 이용해서 화면에서 숨김


label 태그 안에 input 태그를 작성해주면
for 속성을 작성하지 않아도 input과 연결됨

태그에서 특정 속성을 선택하여 디자인하기

input 태그

        input[type=""]형식의 선택자는 특정 type 속성값을 가진
        input 요소를 선택하는데 도움됨
        예를 들어
        input[type="text"] type이 text인 입력필드만 선택돼서 꾸며짐
