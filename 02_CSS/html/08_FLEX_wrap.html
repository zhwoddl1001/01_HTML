        display : flex 가로로 정렬 기존에 있던 display 스타일을 변경했기 때문에 가능

        flex-wrap : 컨테이너 안에 있는 여러개의 아이템이 작은 화면에서도
                    여러줄에 걸쳐 배치되도록 설정
                    반응형 디자인
                    반응형이란 : 화면에서 화면의 가로 크기에 따라 사용자에게 거부감 없는 디자인을 제공하는 것을 반응형
                    본체 모니터 800~ 1400
                    태블릿 700px ~ 500px
                    핸드폰 400px ~ 300px
