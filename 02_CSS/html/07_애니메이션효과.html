transform 속성

        transform : 요소를 이동, 회전, 크기 조절, 뒤집기 등의 변환을 적용하는 데 사용
        1. transform : 요소를 X, Y 축 방향으로 이동
            transform: translate(10px, 20px); → 가로 오른쪽으로 10px 세로 아래쪽으로 20px 이동
                                                숫자 + 3글자 오른쪽 3글자 아래쪽 3글자
                                                숫자 - 4글자 왼쪽 2글자 위 2글자
        transform: rotate(45deg);   =  시계방향 45도 회전
        transform: scale(1.5) = 태그와 태그안에 있는 값 = 요소의 크기를 1.5배 키움
        transform: skew(10deg,5deg) 가로 오른쪽으로 10도, 세로 오른쪽으로 5도 기울이기

        예를 들어
        요소를 오른쪽으로 20px 아래로 10px 이동하고 크기를 1.2배로 키운다
        이것은 transform 기능을 2개 이상 사용할 때 작성하는 방법
        띄어 쓰기로 구분하여 두가지 사용가능

트랜지션과 트랜스폼

      트랜지션은 배경색 글자크기 글자색 가로세로 크기 등 다양한 스타일 속성값에 속도변화를 줄 수 있음

      트랜스폼은 이동하기 기울이기

      상대적으로 트랜지션이 트랜스폼보다 많이 사용
고개를 돌리다.
