확장 파일로 연결한 CSS선택자

반응 선택자

        사용자의 움직임에 반응하여 스타일이 변경되게 하는 선택자속성
        
        선택자 : active → 선택된 요소가 클릭되고 있을 때
                        image 를 이용해서 프로필 사진 미리보기
        선택자: hover → 선택된 요소 위에 마우스가 올라갔을 때
                        button 태그에서 많이 사용
        선택자: focus → 선택된 요소에 초점이 맞춰졌을 때 생기는 선
                        input 태그에서 주로 사용
