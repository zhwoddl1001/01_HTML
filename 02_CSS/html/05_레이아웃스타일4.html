position

        요소 배치(위치) 관련 스타일
        위치를 지정하는 속성

        position: relative(상대적인)
         지정된 요소 내부에 다른 요소가 상대적인 위치를 지정할 수 있도록 기준이 되게 만드는 속성

        position: absolute(절대적인)
         기본 요소 배치 순서를 무시하고 지정된 절대 위치에 요소를 배치
         z-index : 정수
         가장 우선으로 보여지는 디자인의 순서를 지정
         숫자가 적을 수록 겹쳐졌을 때 후순위
         숫자가 클 수록 먼저 보여짐

         z-index : 단독으로 많이 사용하기도 함

          relative, absolute, fixed, sticky 같이 사용됨
           기본숫자 : 0
           -2147483648 : 32bit 기준으로 최소값
           -1 : 다른 요소 아래에 쌓임
            0 : 기본 레벨
            1 : 다은 요소 위에 쌓임
            10: 더 높은 우선 순위를 가지며 더 많은 요소 위에 보여짐
            +2147483648 : 32bit 기준으로 최대값
         position: fixed(고정된)
         항상 고정된 위치에 요소를 배치함(화면이 움직이든 말든 항상 같은 위치)
       
         position: sticky
         원래 요소사 보여지는 화면이라면 position X
         요소가 보여지는 화면을 벗어나면 fixed (고정)
