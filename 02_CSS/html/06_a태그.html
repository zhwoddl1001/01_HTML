하이퍼링크 관련 태그 - a 태그

        하이퍼 링크 태그

        a태그 내부에 속성으로 href 작성하면 글자 색상과 밑줄이 생김 글자색상 기본적
        파란새 - 아직 클릭하지않은 링크색상
        보라색 - 이미 클릭해서 방문한 링크색상
        
        a 태그는 사용자가 특정 행동을 했을 경우를 3가지 경우로 나누어 설정

        a:hover = 마우스를 링크 위에 올렸을 때 스타일 설정
            주로 text-decoration : underline 밑줄 추가 설정
        a:active = 마우스를 클릭했을 때 활성화 되느 스타일
            주로 글자색을 또 한 번 변경하고 클릭시 약간 줄어들거나 커지게 만들어줌

        a:visited
