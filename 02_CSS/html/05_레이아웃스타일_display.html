display 속성

        display 속성 HTML 요소 배치 방식을 설정하는 중요한 css속성
        요소 : <태그>내용 element
        display 주요 값으로
        [block, inline, inline-block, flex, grid, inline-flex]
        모든 태그에는 display가 각 속성에 맞게 기본설정이 되어있음
        display : block = div, h1~h6, p 등 대표적으로 block 설정으로 존재
                예 ) h1 태그를 만들 때 : display:block 주고 
                                        font-weight: bold 주고
                                        font-size:32px 
                                        color:black
                                        기본 설정으로 만들자
                        위 h1 태그처럼 모든 태그에는 각 css가 기본으로 설정이 되어있음
       
        display : inline = span, a, strong 등 대표적으로 inline 설정으로 존재
                예 ) span 태그를 만들 때 : display:inline 주고
                                         font-size:16px 주고
                                         color:black 
                                         기본 설정해서 개발자들이 span 태그를 활용할 수 있게 하자
        
        기본적으로 각 태그에는 기본스타일이 적용되어 있는데
        기본스타일의 틀을 변경하길 원할 경우 css를 이용해서 디자인 변화를 주는 것
        display : inline-block = inline 처럼 한 줄에 표시되지만 이미지, 버튼에 주로 사용
                                img, button 태그가 inline-block 설정

display: flex

        flex 요소는 내부 항목들이 수평 또는 수직으로 유연하게 정렬
        inline과 flex 차이  
        inline = 가로 세로 조정X
        flex   = 가로 세로 조정O

display : grid

행과 열로 내부 항목 정렬
        주로 이미지나 웹툰처럼 틀과 형식으로 규칙적인 정렬이 필요할 때
        많이 사용

        grid-template-columns: 1fr 1fr; 
            fr : fraction(부분, 일부)
            그리드 내에서 사용 가능한 공간을 비율로 나누는 단위
            1fr : 그리드의 사용 가능한 공간을 같은 비율로 나눠서 차지하라는 의미
            1fr 1fr : 두 개의 열이 동일하게 공간을 나누어 차지

            1fr 2fr : 첫 번째 열은 가로 전체 공간의 3분의 1을 차지
                      두 번째 열은 가로 전체 공간의 3분의 2을 차지
        gap: 10px;
            그리드 셀 사이에 10px 간격을 추가
            그리드 안에서 자동으로 10px 간격을 주고 자동배치
