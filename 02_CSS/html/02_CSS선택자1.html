CSS 선택자 1

태그 선택자

        -현재 문서 내에 특정 태그를 모두 찾아 선택하는 선택자
        [작성법]
        
            태그면{
                css코드;
            }

id(#)태그 선택자

        HTML 문서 내에서 id가 일치하는 요소를 선택하는 선택자
        [작성법]
        
            #id이름{
                css코드;
            }

class.선택자

        -html 문서 내에서 class속성 값이 일치하는 모든 요소를 선택하는 선택자
        style에서는 class를 주로 사용
        id 값 주로 자바 스크립트
        class를 디자인에서 필수로 사용하기(개발자 간의 약속)
        작성법
        

            .class이름명{
                class 고드;
            }
