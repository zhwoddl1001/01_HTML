        -css 확장파일로 따로 작성해주지 않을 경우
        헤드 태그(head 태그) 사이에 style 태그를 작성
        body나 foot 과 같은 태그에 작성해도 되지만
        style 태그 같은 경우 주로 head 태그 안에 작성
    **css 코드를 작성하는 방법**
  • head 태그에 style 태그를 작성한 후 내부에 css 코드 작성

    • 예 - head 태그 style 태그 h1{color:red;}
  • 원하는 요소에 직접 style 속성을 이용해서 css 코드를 작성
