영역 관련 태그

div 태그

            -문서(document) 코딩(html) 내에서 구역이나 컨테이너를 지정하는데 사용

            -p 태그와 다른점 p태그는 글자만 구역

            -div 태그는 그림 동영상 등 여라가지를 태그 틀 안에 담을 수 있음

            -div 태그를 나가면 자동으로 줄바꿈 처리가 됨

span 태그

            -이어서 작성 가능한 틀을 제공하는 태그

div 와 스판 비교하기

첫 번째 div
두 번째 div
세 번째 div
첫 번째 span 두 번째 span 세 번째 span

block 형식

        -화면(공간)을 수직 분할(세로 화면을 가로로 잘라서 층을 나누는 행위)
        -가로 한 줄 전체를 차지하기 때문에  / 태그를 닫는 순간 줄바꿈이 이루어짐
        width, height 와 같은 크기 지정 속성 사용 가능

        -block 형식의 태그(태그가 닫히면 자동 줄바꿈 이루어지는 태그들)
         h1 ~ h6 (제목), p/pre (문단), hr(수평선) div(영역 분할), 시맨틱 태크 (div 태그에 이름을 넣는 태그)

inline 형식 태그

        -화면을 수평 분할(가로 화면을 세로로 잘라서 열을 나눔)
        -가로에서 큰 공간을 차지하지 않고 사용하고 싶은 만큼만 공간을 차지한 다음에 옆자리를 내어줌
        -width, height와 같은 크기 지정 속성 사용 불가
        -inline 형식의 태그
        b(굴게), strong(굵게), i(기울이기), mark(글배경 색칠하기), span(영역 분할)

크기 지정 가능 여부 차이점

  • block : 크기 지정 가능
  • inline : 크기 지정 불가능

inline - block 형식

            -inline의 특징 중 수평 분할 (세로로 자름)
            -block의 특징 중 크기 지정 가능

            -대표적인 태그 : img 태그, iframe 태그


            *iframe 태그
            -웹 문서(html) 내부에 또 다른 웹  문서를 추가하는 태그
