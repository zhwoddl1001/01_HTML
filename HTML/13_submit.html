버튼 형식 태그



input type = "submit"

        
        -form 태그에 작성한 내용을 제출하는 버튼 역할을 함
        버튼을 누르면 폼 데이터가 action에 지정된 제출 위치로 전송
        
        -텍스트와 아이콘을 함께 넣기 어렵고, 꾸미는데 제한적임

버튼 관련 태그

        -기본 타입이 submit으로 form 태그에 작성한 내용을 제출하는 버튼 역할을 함
         type을 submit 로 작성하지 않아도 제출하는 것이 기본설정
        이기 때문에 작성을 하지 않아도 됨
       
        -종류 : submit(제출), button(클릭 후 제출이나 초기화 기능 이외 설정), reset(초기화)
        
        -이미지나 아이콘을 넣어 꾸밀 수 있음
