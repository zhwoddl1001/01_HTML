form 태그 / input 태그

form 태그

        -입력 양식을 작성하는 공간
        -내부에 입력된 값을 지정된 곳으로 제출하는 역할

        [속성]
        1 action       : 제출될 곳을 지정하는 속성(제출할 주소 설정)
        2 method(방법) : 설정된 곳으로 데이터를 전달하는 방식
        3 name         : 이름 지정하는 속성 (JavaScript 함께 사용)
        4 target       : 제출된 페이지를 현재/새 탭으로 열기 설정

input 태그

        -웹 문서에서 사용자가 입력할 수 잇는 양식을 제공하는 태그
        -많은 type(속성)이 존재하고, input의 변형 형태도 다수 존재

        *input 태그에는 name 속성이 필수로 작성되어야 함! 꼭은 아니지만 작성하는 경우 (80% 이상)가 많음
        왜냐하면 name 이용해서 이름끼리 서로연결
